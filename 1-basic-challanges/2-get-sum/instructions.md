# Challenge: Get sum

Write a function called `getSum` that takes in two numbers and returns the sum of those two numbers.

### Constraints

- The function must return a number

## Solutions

<details>
  <summary>Click For Solution</summary>

```JS
function getSum(a, b) {
  return a + b;
}
```
